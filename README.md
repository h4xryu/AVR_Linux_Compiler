# AVR_Linux_Compiler

## How to install
    $ sudo apt update
    $ sudo apt-get upgrade
    $ sudo apt install qtcreator
    $ sudo apt-get install libqt5serialport5-dev
    $ sudo apt install qt5-qmake
    $ sudo apt install tilda  
    $ make avr_linux_compiler

## How to run
    $ ./avr_linux_compiler
